# Security Policy

## Supported Versions

The `sravya` branch and the latest tagged release receive security fixes during the hackathon review period.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately to the project maintainers before opening a public issue. Include reproduction steps, affected files, expected impact, and any suggested mitigation.

## Security Scope

NeuroNote is designed as an offline-first local application. Core AI processing must run through local runtimes such as Ollama, and uploaded study material should remain on the user's device.
