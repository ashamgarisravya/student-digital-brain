# Changelog

## 0.3.1 - 2026-06-29

- Switched repository licensing to AGPL-3.0-or-later.
- Added required user, agent, security, conduct, Docker, and environment documentation.
- Added spec-kit structure and offline PDF workflow specification.
- Expanded CI with format, lint, type check, test, coverage, security, and audit stages.
- Added coverage threshold, secret scanning baseline, and additional quality tool configuration.

## 0.3.0 - 2026-06-28

- Added PDF-first source metadata for uploaded documents.
- Added page-aware keyword search for newly processed PDFs.
- Added revision flow: mind map, definitions, questions, diagrams, and daily process.
- Added quiz modes for MCQ, very short, short, long, and definition answers.
- Added semantic-style answer evaluation for written quiz answers.
- Added GitLab CI with real compile, lint, type, test, audit, and security checks.
- Added local audit script for offline PDF extraction, SQLite schema, no-cloud scan, and repo metadata.
- Added pre-commit hooks.

## 0.2.0 - 2026-06-28

- Added SQLite-backed backend MVP.
- Added PyMuPDF PDF parsing, Tesseract OCR path, Whisper.cpp path, llama.cpp path, search, quiz, revision, and dashboard APIs.
- Added unit and end-to-end backend tests.

## 0.1.0 - 2026-06-28

- Added Streamlit frontend MVP with dashboard, uploads, search, knowledge graph, quiz, revision, progress, and settings pages.
